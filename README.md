## Memo

### エンディアン(バイト単位で見た時のデータの並び順)

- 予備知識

|用語|意味|
|:--:|:--:|
|データ|-|
|ビット|情報の機本単位。0か1で表現される|
|バイト|1byte = 8bit|
|エンディアン|複数のバイトで構成されるデータを取り扱うときの並び順のこと|

- ビッグエンディアンとリトルエンディアン

|用語|意味|
|:--:|:--:|
|ビッグエンディアン|上位バイトから下位バイトの順に取り扱う方式|
|リトルエンディアン|下位バイトから上位バイトの順に取り扱う方式|
